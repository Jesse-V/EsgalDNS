EsgalDNS (alpha) - Distributed DNS for Tor Hidden Services

A distributed, Tor-powered, and fully confirmable DNS system for Tor hidden services. The goal of this project is to associate .tor domain names with existing .onion addresses. This will significantly increase the usability and accessibility of Tor hidden services. The project avoids any central authority overseeing the system and is built with security, privacy, and anonymity in mind.

This is still under development and is nowhere near ready for deployment. Stay tuned!
